# VideoBox API

## Tecnologias

- Node.js + Express
- MongoDB para banco de dados
- Joi para validações
- Bcrypt para hash de senhas
- UUID para controle de sessões
- Jest + Supertest para testes

## Instalação e execução

git clone https://github.com/thiagoss2000/VideoBox.git

npm install
npm run dev
